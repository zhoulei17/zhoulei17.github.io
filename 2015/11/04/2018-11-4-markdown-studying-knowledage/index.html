9、代码

单行代码：代码之间分别用一个反引号包起来

System.out.println("hello world")

代码块：代码之间分别用三个反引号抱起来，且两边的反引号独占一行

public class A
{
    public static void main(String[] args)
    {
        String a = "abc";
        System.out.println(a);
        if ( a != null)
        {
            System.out.println("hello world!");
        }
    }
}
